# Hello GAS

> Hello Google Apps Scripts

## Installation

```bash
git clone git@github.com:nando-experiments/hello-gas.git
cd hello-gas/
npm install
```

## Dependecies

 - [@google/clasp](https://github.com/google/clasp) - Develop Apps Script Projects locally
 - [@types/google-apps-script](https://github.com/DefinitelyTyped/DefinitelyTyped) - TypeScript definitions for google-apps-script
